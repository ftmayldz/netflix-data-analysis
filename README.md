# netflix-data-analysis
Performed exploratory data analysis on Netflix dataset using Python (pandas, matplotlib, seaborn) to identify content trends by year, country, and genre.
# Netflix Data Analysis

## Problem Statement
Analyze Netflix content to understand trends by year, country, and genre.

## Dataset
Netflix Movies and TV Shows dataset from Kaggle.

## EDA Steps
- Data cleaning
- Missing value handling
- Exploratory visualizations

## Key Insights
- Movies dominate over TV Shows
- USA produces most content
- Content increased after 2015
- Drama is most common genre

## Conclusion
Basic exploratory analysis provides insights into Netflix content strategy.
