Exploratory Data Analysis project on Netflix dataset to uncover trends by year, country, and genre.

# netflix-data-analysis
Performed exploratory data analysis on Netflix dataset using Python (pandas, matplotlib, seaborn) to identify content trends by year, country, and genre.
# Netflix Data Analysis

## Problem Statement
Analyze Netflix content to understand trends by year, country, and genre.

## Dataset
Netflix Movies and TV Shows dataset from Kaggle.

## EDA Steps
- Data cleaning
- Missing value handling
- Exploratory visualizations


## Key Insights

- Content production increased significantly after 2015
- United States produces the highest number of titles
- Drama is the most common genre
  

## Tools Used 
-Python 
-Pandas 
-Matplotlib
-Seaborn 
-Jupyter Notebook 


## Conclusion

This analysis explored Netflix content trends across years, countries, and genres.
Key insights were derived to understand platform growth and content strategy.
